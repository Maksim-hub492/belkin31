To-Do List на Vue.js
Простое приложение для управления задачами, созданное с использованием Vue.js 3.

Функциональность
 Добавление новых задач
 Отметка задач как выполненных
 Удаление задач
 Фильтрация задач (Все/Активные/Выполненные)
 Автоматическое сохранение в Local Storage
 Статистика по задачам
Технологии
Vue.js 3
HTML5
CSS3
Local Storage API
Запуск проекта
Способ 1: Прямое открытие в браузере
Скачайте файл index.html
Откройте его в любом современном браузере
Способ 2: Через локальный сервер (рекомендуется)
Установите простой HTTP-сервер:
# Если у вас установлен Node.js
npm install -g http-server

# Или используйте Python
python -m http.server 8000
Запустите сервер в папке с проектом:
http-server
# или
python -m http.server 8000
Откройте в браузере: http://localhost:8080 или http://localhost:8000
Способ 3: Использование Live Server в VS Code
Установите расширение "Live Server" для VS Code
Правой кнопкой мыши на файле index.html → "Open with Live Server"
Особенности реализации
Чистый Vue.js: Без дополнительных зависимостей кроме Vue 3
Адаптивный дизайн: Работает на мобильных устройствах и компьютерах
Persistent Storage: Задачи сохраняются между сессиями
Интуитивный интерфейс: Простой и понятный UX
Разработка
Для внесения изменений:

Откройте index.html в текстовом редакторе
Внесите необходимые правки в HTML, CSS или JavaScript код
Сохраните и обновите страницу в браузере
Приложение готово к использованию и не требует сборки или компиляции.
