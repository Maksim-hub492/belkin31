To-Do List на Vue.js
Простое приложение для управления задачами, созданное с использованием Vue.js 3.

Функциональность
 Добавление новых задач
 Отметка задач как выполненных
 Удаление задач
 Фильтрация задач (Все/Активные/Выполненные)
 Автоматическое сохранение в Local Storage
 Статистика по задачам
Технологии
Vue.js 3
HTML5
CSS3
Local Storage API
Запуск проекта
Способ 1: Прямое открытие в браузере
Скачайте файл index.html
Откройте его в любом современном браузере
Способ 2: Через локальный сервер (рекомендуется)
Установите простой HTTP-сервер:
# Если у вас установлен Node.js
npm install -g http-server
